在尼姆遊戲中，兩個玩家輪流從不同的堆中取走石頭。在每一步中，玩家都可以選擇任何非空的石頭堆，並從堆中取走至少一塊石頭。取走最後一塊石頭的玩家獲勝。實作一個函數來計算遊戲中的最佳策略。

任務

編寫一個互動程式來玩尼姆遊戲。實現一個函數 \texttt{nim} 來計算在給定的遊戲狀態中的最好的一步。

你的函數： \texttt{nim()}
Pascal: \texttt{procedure nim(n: longint; var a: array of longint)}
C/C++: \texttt{void nim(int n, int a[])}
$N$ ($2 \leq N \leq 100$) 是一個整數，代表石堆的數量。 $a$ 是一個有 $N$ 個整數的數組，其中 $a_i$ ($1 \leq a_i \leq 100$) 代表第 $i^{th}$ 個堆中石頭的數量。

你的函數應呼叫函數 \texttt{answer(int i,int x)} 一次來傳回答案。

Grader 的函數： \texttt{answer()}
Pascal: \texttt{procedure answer(i, x: longint)}
C/C++: \texttt{void answer(int i, int x)}
呼叫 \texttt{answer} 表示答案是在第 $i^{th}$ 個堆中 ($0 \leq i^{th} < N$) 取走 $x$ ($1 \leq x \leq a[i]$) 塊石頭。

對於所有測試數據，如果你使用最佳策略，保證你一定能勝出。
