Given the coordinates of a $N$-sided polygon in anti-clockwise order. Please write a program to calculate the area of the polygon.

Input

The first line contains an integer $N$ ($3 \le N \le 100$).

Each of the following $N$ lines contains two integers $X$ and $Y$ ($-1000 \le x, y \le 1000$), the $x$- and $y$-coordinates of the vertex.

Output

Output the area of the polygon corrected to three decimal places.
