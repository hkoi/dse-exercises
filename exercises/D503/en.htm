Write a program to output a simple, textual bar chart to a text file. You will be given the required height of the 6 bars.

Input

Read the input from standard input (keyboard). The input consists of 6 integers in one line. The $i^{th}$ integer is the required height of the $i^{th}$ column from the left. The integers are between 0 and 9 inclusive.

Output

Output the bar chart to the file \texttt{chart.txt}. Please refer to the sample below for the exact format.

The chart area is 10 lines high and 25 columns wide, and is surrounded by a frame made of characters \texttt{-}, \texttt{|} and \texttt{+}. The background should be painted with the space character \texttt{ }. Each bar should be 3 columns wide and should be painted with the character \texttt{#}. Bars should be separated by one column.
